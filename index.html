Método base

P·C·T·F·R

O método único que serve para qualquer Agent. 5 elementos que transformam qualquer pedido em resultado profissional.

P
Papel
Quem é a IA? Define especialidade e foco antes de qualquer tarefa.
C
Contexto
Qual a situação? Quem vai ler, usar ou receber o resultado.
T
Tarefa
O que exatamente? Específico com número e forma definidos.
F
Formato
Como entregar? Estrutura exata do output esperado.
R
Restrições
O que evitar? Remove erros antes de ocorrer.
❌
Evite

Prompt Fraco

Entrega o mínimo porque pede o mínimo.

"Me faz um post sobre alimentação saudável"

Resultado: post genérico com "alimentação equilibrada" e "hábitos saudáveis". Sem voz, sem conversão.

✅
Use

Prompt Forte (P·C·T·F·R)

Cada elemento faz a IA trabalhar com intenção.

Você é [NUTRICIONISTA] especialista em [EMAGRECIMENTO].
(P) Papel com especialidade e foco definidos.

Meu público são mulheres [28–45 anos] que trabalham muito.
(C) Contexto: quem vai ler esse conteúdo.

Crie um carrossel de [7 slides] sobre lanches para não sentir fome à tarde.
(T) Tarefa específica com número definido.

[SLIDE 1]: headline impactante.
[SLIDES 2–6]: dica + 2 linhas. [SLIDE 7]: CTA.
(F) Formato exato entregue.

Sem termos técnicos. Tom [leve e motivador].
(R) Restrições claras — o que não fazer.

Referência

Estrutura dos 7 Slides

Função de cada slide e o que não pode faltar em nenhum.

Slide Função Regra
01 Gancho visual Cliffhanger — máx. 8 palavras. Promete o que vem.
02–05 1 ideia por slide Headline 6 palavras + 2 linhas. Valor real por slide.
06 Resumo ou Bônus Síntese ou dica exclusiva. Motivo para salvar.
07 CTA direto 1 ação — salvar / comentar / agendar. Nunca dois CTAs.

Montagem Final

Como montar no Canva ou PowerPoint

Com as imagens geradas no ChatGPT, monte sua apresentação em poucos minutos.

# Passo
1 Gere os slides com o prompt acima no ChatGPT — peça um slide por vez
2 Baixe cada imagem gerada (clique com o botão direito → Salvar imagem)
3 Canva: Crie uma apresentação → insira cada imagem como fundo do slide (Uploads → arraste para o slide)
4 PowerPoint: Inserir → Imagens → selecione os arquivos → ajuste para preencher o slide inteiro
5 Adicione transições, ajuste a ordem e exporte em PDF ou apresente ao vivo
